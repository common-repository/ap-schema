= 2.0.0 =

* Complete plugin overhaul, re-coded from the ground up.
* Shortcode system to allow review boxes anywhere in your post/page.
* Multiple review boxes allowed.
* 8 different types of reviews: Book, Event, Movie, Organisation, Person, Product, Recipe, Review.
* Multiple style types: Basic (no styling), Green, Light Blue, Orange, Pale Yellow, Red-pink, Silver Grey, Turquoise.
* Multiple star types: Yellow Star, Green Star, Blue star, Red Heart, Green Tick, Yellow lightning, Black Skull, White Skull.
* Shortcodes can be added to Text Widgets allowing reviews in the sidebar.
* Translation ready
